# Keyboard Shortcuts

- `[SPACE]` (EN, hold for 3 seconds): Show remaining battery and last sync time.
- `[NEW]` + `[SHIFT]`: EN 한국어 전환
  - `~`: 한국어 일때 한영 전환
- right `[NEW]` + `[SHIFT]`: Check firmware update
