# 선택 이유
Deno는 Ryan Dahl의 새로운 프로젝트, TypeScript 기반. Rust 구현은 나에게는 그리 큰 잇점은 아니다.

# 기타
- `node`: js 실행
- `deno`: ts 실행
- `ts-node`: `npm install -g ts-node`로 설치. `tsc`가 변환 후 `node`로 실행. 원래 `tsc`는 js 변환 역할만 수행.
